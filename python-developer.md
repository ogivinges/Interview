```python
class Animal:
    def speak(self):
        return "Animal говорит"

class Cat(Animal):
    def speak(self):
        return super().speak() + "Кошка говорит"

class Dog(Animal):
    def speak(self):
        return super().speak() + "Собака говорит"
  
class MyAnimal(Cat, Dog):
    def speak(self):
        return super().speak() + "Мое животное говорит"
  
a = MyAnimal()
print(a.speak())
```

Выведется: Animal говорит Кошка говорит Собака говорит Мое животное говорит
Можно узнать порядок через print(MyAnimal.mro())
